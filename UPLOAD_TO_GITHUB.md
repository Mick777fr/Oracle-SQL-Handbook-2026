# Mettre ce pack sur GitHub

Le depot etant vide, la methode la plus simple est :

1. Ouvrir le depot GitHub.
2. Cliquer sur **Add file** puis **Upload files**.
3. Decompresser le ZIP sur votre appareil.
4. Selectionner tous les fichiers et dossiers contenus dans le dossier racine.
5. Ajouter le message de commit : `Initialisation du handbook et ajout du chapitre 1 v0.1`.
6. Cliquer sur **Commit changes**.

Important : importez le contenu du dossier, pas le dossier parent lui-meme, afin que `README.md` se trouve a la racine du depot.
