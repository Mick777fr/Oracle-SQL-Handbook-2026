# Chapitre 1 - Exercices

## Questions

1. Distinguez SQL, base de donnees et SGBD.
2. Donnez quatre risques lies a la duplication de donnees.
3. Comparez une feuille de calcul et un SGBD pour une application multi-utilisateur.
4. Expliquez le caractere declaratif de SQL.
5. Resumez l'apport du modele relationnel.

## Corrige synthetique

1. SQL est un langage ; la base contient les donnees ; le SGBD gere leur stockage et leur acces.
2. Incoherences, surcout de maintenance, erreurs de mise a jour, gaspillage d'espace.
3. Le SGBD offre transactions, droits fins, contraintes et concurrence controlee.
4. L'utilisateur decrit le resultat souhaite sans imposer le chemin physique.
5. Il fournit une representation logique tabulaire et une manipulation fondee sur les ensembles.
